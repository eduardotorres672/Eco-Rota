Uma aplicação interativa de Quiz desenvolvida em **Python** que desafia a inteligência e a velocidade do usuário. O projeto implementa lógica de gamificação, controle de tempo e diferentes níveis de dificuldade em um ambiente de terminal elegante.

---

## 📝 Sobre o Projeto

O **Python Quiz Pro** é um simulador de conhecimentos gerais que avalia não apenas a precisão das respostas, mas também a agilidade mental. Com um banco de perguntas categorizado, o software oferece uma experiência progressiva de aprendizado e entretenimento.

### ✨ Funcionalidades Principais

*   **Níveis de Dificuldade**: Escolha entre níveis **Fácil**, **Médio** e **Difícil**, cada um com seu próprio conjunto de desafios.
*   **Cronômetro de Reação**: Monitoramento em tempo real (em segundos) de quanto tempo o usuário leva para responder cada questão.
*   **Feedback Inteligente**: Sistema de pontuação dinâmica que fornece mensagens personalizadas baseadas no percentual de acerto final.
*   **Processamento de Texto**: Validação de respostas *case-insensitive* para evitar erros de digitação simples.
*   **Loop de Jogo**: Opção de reiniciar o desafio instantaneamente ao final de cada rodada.

---

## 🛠️ Tecnologias e Lógica

*   **Python 3.x**: Linguagem principal.
*   **Module `time`**: Utilizado para o cálculo de latência de resposta e precisão temporal.
*   **Estruturas de Controle**: Implementação de loops `while` e condicionais aninhadas para um fluxo de jogo ininterrupto.
*   **Dicionários e Listas**: Organização eficiente do banco de dados de perguntas.
